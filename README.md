# TasteHub

---

프로젝트 개요

"TasteHub"는 사용자가 여러 음식을 비교하면서 선호도를 확인하고, 최종적으로 하나의 우승 음식을 가려내는 재미있는 게임형 서비스입니다. 사용자는 두 개의 음식을 번갈아 선택하며, 자신이 가장 선호하는 음식을 찾아가는 과정을 즐길 수 있습니다. 이 서비스는 친구들과 결과를 공유하거나 다양한 음식 테마를 경험하며 새로운 음식을 발견하는 기회를 제공합니다.

주요 기능

음식 선택 및 비교: 다양한 음식을 두 개씩 비교해 선호하는 음식을 선택.

커스터마이징: 사용자가 직접 음식 리스트를 추가하거나 주제를 선택할 수 있는 기능 제공.

결과 공유: 게임 결과를 소셜 미디어에 공유할 수 있는 기능 제공.

통계 제공: 사용자들의 선택 데이터를 바탕으로 인기 음식 순위 및 선호도 통계 제공.
